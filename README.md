# Python Blockchain
Python implementation of blockchain technology.

In ```blockchain.py``` is the base blockchain class, and the structure of each block.

```proof_of_work_concept.py``` demonstrates how a simple Proof of Work (PoW) algorithm works, using trailing 0's as an example.

However, that actual implementation mines hashes that begin with 1234.



Additionally, a server implementation is set up to demonstrate blockchain and transaction logic when calls are made to mine a new block.
